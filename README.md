# Mission-recyclage-

## Description
Jeu interactif en Python destiné à sensibiliser les enfants 
de 6 à 10 ans au tri sélectif des déchets.
Le joueur lance différents types de déchets dans la bonne 
poubelle grâce à un calcul de trajectoire.
Des instructions sur le recyclage sont affichées avant de 
jouer, et des rappels apparaissent après chaque erreur.



## Problématique
Comment donner envie aux enfants d'agir concrètement pour la planète, à travers un jeu qui reproduit les vrais gestes du recyclage ?
